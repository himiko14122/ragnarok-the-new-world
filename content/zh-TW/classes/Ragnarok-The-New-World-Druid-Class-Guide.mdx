# 德魯伊職業指南 - 三種形態大師

德魯伊是《Ragnarok The New World》中最具多功能性的職業，擁有獨特的三種形態變身系統，可以在狼人、鳥人和人類秘術師形態之間轉換。每種形態提供不同的戰鬥角色：狼人形態進行近戰傷害、鳥人形態進行遠程攻擊，而人類秘術師形態則進行法術施放。這種多功能性使德魯伊能夠在緊急情況下填補任何角色，但為了優化三種形態所需的三重裝備投資，使這個職業成為遊戲中資源需求最高的職業之一。本指南涵蓋每種形態的最佳配置、屬性分配策略，以及管理德魯伊複雜性的實用建議。

德魯伊沒有二轉職業，而是通過形態專屬技能樹和在戰鬥中切換三種形態的能力來獲得力量。變身機制與槍手的武器切換系統在概念上有相似之處，但德魯伊的形態差異更大，每種形態都需要完全不同的屬性優先級和裝備配置。

## 三種變身形態

### 狼人形態

狼人形態將德魯伊轉變為強大的近戰攻擊者，擁有增強的身體屬性和利爪攻擊。此形態是德魯伊在近距離遭遇戰和需要近戰範圍傷害的刷怪場景中的主要傷害輸出來源。

狼人的攻擊速度快，造成由 STR 和 AGI 加成的物理傷害。此形態獲得攻擊速度和暴擊率的被動加成，使狼人形態在持續近戰 DPS 方面表現出色。主要限制是需要處於近戰範圍，這使德魯伊暴露在敵方攻擊下，而鳥人形態則可通過距離來避免。

### 鳥人形態

鳥人形態通過風屬性投射物提供遠程攻擊能力。此形態提供安全的攻擊距離和中等持續傷害，非常適合保持距離對生存至關重要，或目標移動性強且難以接近的戰鬥。

鳥人的攻擊造成風屬性傷害，對特定怪物類型有效，但與其他遠程職業的無屬性傷害相比，其多功能性有限。此形態的傷害輸出低於獵人的 ADL 配置，但以機動性優勢和在需要時快速變身為狼人進行近戰爆發的能力作為補償。

### 人類秘術師形態

人類秘術師形態提供具有範圍傷害技能的法術施放能力。此形態是德魯伊對團隊內容的貢獻，提供範圍傷害來補充巫師的範圍攻擊，同時提供在關鍵時刻變身為治療形態的選項。

人類秘術師的範圍法術威力不如巫師的專屬範圍技能庫，但在團隊環境中提供有意義的補充傷害。此形態的主要價值在於其多功能性：能夠在群怪階段施放範圍技能，然後變身為狼人進行 Boss 爆發傷害，再根據需要變身為輔助角色。

## 屬性分配

### 通用德魯伊配置

由於德魯伊使用三種具有不同屬性需求的形態，因此支持所有形態的通用屬性分配至關重要。挑戰在於狼人形態依賴 STR 和 AGI，鳥人形態依賴 DEX 和 AGI，而人類秘術師形態依賴 INT 和 DEX。沒有任何一種屬性分配能同時優化這三種形態。

| 屬性 | 數值 | 主要形態 | 備註 |
|------|-------|-------------|-------|
| STR | 50-70 | 狼人 | 近戰傷害加成 |
| AGI | 60-80 | 狼人、鳥人 | 兩種物理形態的攻擊速度 |
| VIT | 30-50 | 全部 | 通用生存能力 |
| INT | 40-60 | 人類秘術師 | 法術傷害和 SP |
| DEX | 50-70 | 鳥人 | 遠程命中率和傷害 |
| LUK | 20-30 | 狼人 | 暴擊率補充 |

這種通用配置能在所有三種形態中提供功能性表現，但不會在任何特定形態中特別出色。AGI 的投入通過攻擊速度使兩種物理形態受益，使其成為最有效的通用屬性。VIT 為所有形態提供基礎生存能力，而適中的 STR、INT 和 DEX 則為每種形態提供可用的傷害加成。

### 專精形態配置

主要使用一種形態的玩家應更積極地將屬性分配給該形態的需求。專注於狼人形態的德魯伊應最大化 STR 和 AGI，並最小化 INT，而專注於人類秘術師的德魯伊應優先考慮 INT 和 DEX，並最小化 STR。權衡是非優先形態的效果會顯著降低。

## 裝備策略

### 形態專屬裝備

德魯伊的裝備挑戰在於每種形態受益於不同的裝備屬性。狼人裝備需要力量、敏捷和物理傷害加成。鳥人裝備需要靈巧和遠程傷害加成。人類秘術師裝備需要智力和縮短詠唱時間。這意味著一個完全優化的德魯伊需要三套完整的裝備，投資是大多數其他職業的三倍。

### 預算優化

對於預算有限的玩家，專注於一個主要形態並全力投資其裝備，然後隨著資源允許逐步建立次要形態的裝備。這篇[安全精煉指南](/refining/safe-refining-guide)提供了在有限預算下行之有效的精煉策略。應優先精煉所有形態通用的裝備，如斗篷、鞋子和飾品。

## FAQ

### 哪個德魯伊形態最強？

狼人形態提供最高的單體DPS，是大多數玩家的推薦主要形態。鳥人形態提供安全的遠程攻擊，而人類秘術師則提供範圍傷害和治療功能。大多數德魯伊玩家專精一種形態，同時保持其他形態的基本功能。

### 德魯伊對無課玩家來說太貴了嗎？

由於需要三套裝備，德魯伊是資源需求最高的職業之一。無課玩家可以通過專精一種形態並建立一套裝備來有效遊玩德魯伊，但這個職業的全部潛力需要投入難以在不課金的情況下維持。這篇[無課職業強度表](/tier-list/f2p-tier-list)據此對德魯伊進行了排名。

### 戰鬥中變形如何運作？

變形大約需要1-2秒，可以在戰鬥中執行。變形會將你的主動增益和技能欄重置為新形態的配置。請在戰鬥的自然空檔期規劃你的變形，例如清完一波怪物後或在MVP的轉階段期間。

### 德魯伊能在隊伍中取代牧師嗎？

德魯伊的自然祝福能提供群體治療，但明顯弱於牧師的專用治療技能組。在需要持續治療和復活能力的挑戰性內容中，德魯伊無法取代牧師。然而，在大型隊伍中，德魯伊能提供輔助治療，減輕主牧師的負擔。

### 德魯伊與槍手相比如何？

德魯伊和槍手都通過形態/武器切換提供戰鬥多樣性。槍手在DPS角色內維持兩種武器以獲得射程靈活性，而德魯伊則提供橫跨DPS、遠程和輔助角色的三種形態。德魯伊更全面但較不專精，而槍手更專注於DPS適應性。詳見[全職業概覽](/classes/all-classes-builds)以獲得詳細比較。
