## 職業自由系統完整指南

職業自由系統是 Ragnarok The New World 與傳統永久性職業選擇 MMORPG 的決定性差異。此系統允許任何角色在八種基礎職業之間自由切換，無需為了不同定位而創建替代角色。了解職業自由系統的運作方式、哪些內容會在職業間轉移，以及如何管理多職業成長，將使你的遊戲體驗從單一職業轉變為多功能、適應性強的玩法，最大化在米德加爾特的所有活動效益。

職業自由系統從根本上改變了你處理職業選擇和長期規劃的方式。與其永遠投入單一職業，你將發展出一個能滿足不同目的的職業組合。一個角色可以在同一個遊戲階段中，作為騎士坦 MVP、切換成商人進行經濟活動，再轉變為獵人進行單人農怪。這種靈活性伴隨著特定的規則和成本，本指南將詳細說明。

## 職業自由系統如何運作

### 核心機制

當你透過職業自由系統切換職業時，會同時發生幾件事。你的基礎等級保持不變，因為基礎等級代表你角色的整體進度，與職業無關。你的職業等級會重置為 1，需要從頭開始提升新職業的職業等級。你的素質點數會重置，讓你能夠為新職業重新分配適合的屬性。

你先前職業的專屬裝備會保留在倉庫中，但直到你切換回去之前，新職業無法裝備。每個職業都有自己的裝備限制，而職業自由系統會保留所有裝備，在切換過程中不會有任何損失。

### 哪些內容會在職業間保留

| 項目 | 是否保留 | 備註 |
|---------|-------------|-------|
| 基礎等級 | 是 | 所有職業永久共享 |
| 任務進度 | 是 | 已完成的主線和支線任務 |
| 倉庫物品 | 是 | 所有物品都會保留 |
| 裝備 | 是 (在倉庫中) | 職業專屬裝備會被儲存 |
| Zeny 和貨幣 | 是 | 所有職業共享 |
| 職業等級 | 否 | 新職業重置為 1 |
| 技能 | 否 | 每個職業必須重新學習 |
| 素質點數分配 | 否 | 重置以便重新分配 |
| 職業轉職 | 否 | 每個職業必須重新進行 |

### 哪些內容不會保留

職業等級和技能分配是職業專屬的。當你切換到一個新職業時，你會從職業等級 1 開始，沒有任何已學習的技能。你必須透過戰鬥和任務來獲得職業經驗，以提升新職業的等級並解鎖其技能。二轉職業轉職必須為每個職業獨立完成，需要為每個你想發揮全部潛力的職業重做轉職任務。

## 職業切換流程

### 進入切換介面

職業切換介面可透過角色選單或拜訪普隆德拉的特定 NPC 來進入。該介面會顯示所有可用職業、你目前在每個職業上的進度，以及切換的成本。選擇你想切換的職業，確認轉換，你的角色就會轉變為新職業，並擁有相應的外觀和起始技能。

### 切換成本

職業切換需要消耗資源，成本會根據你維持高職業等級的職業數量而增加。第一次切換通常是免費或非常便宜，以鼓勵玩家嘗試不同職業。後續的切換會產生逐漸增加的 Zeny 成本，作為一種軟性限制，控制玩家能有效維持的職業數量。

成本結構鼓勵玩家專注於兩到三個職業，而不是試圖維持全部八個。每個額外的職業都會倍增你的裝備、精煉和技能投資成本，使得全職業覆蓋對大多數玩家來說不切實際。

## 最佳多職業策略

### 雙職業策略

最常見且最具成本效益的方法是維持兩個互補的職業。熱門組合包括：

- 騎士和法師：組隊內容坦職，法師用於農怪
- 獵人和商人：戰鬥內容輸出，商人用於經濟
- 劍士和服事：單人坦職，團隊治療支援

雙職業策略能最大限度地降低裝備成本，同時為大多數內容類型提供角色定位覆蓋。你的主要職業負責推進進度，而次要職業則涵蓋主要職業較弱或需要不同定位的活動。

### 三職業策略

有野心的玩家會維持三個職業以獲得全面覆蓋。此策略需要顯著更多的資源投入，但能在所有內容類型中提供最大的靈活性。

常見的三職業配置包括一個用於農怪和單人內容的輸出職業、一個用於組隊活動的坦克或輔助職業，以及一個用於經濟活動的商人。我們的[職業構建區塊](/classes/all-classes-builds)提供了針對每個定位的最佳化構建，能最大限度地減少重疊並最大化效率。

### 經濟職業整合

許多玩家將商人職業整合到他們的多職業組合中，專門為了經濟利益。商人的露天商店技能允許透過[擺攤交易](/zeny-farming/stall-trading-guide)獲得被動收入，而鐵匠的鍛造技能可以生產能賣出高價的武器。透過在市場活躍時段切換為商人，並在進行內容時切回戰鬥職業，你可以同時最大化進度和收入。

## 跨職業的素質與技能管理

### 切換時的素質重置

每個職業都需要不同的素質分配以達到最佳表現。劍士優先考慮力量和體質，法師需要智力和靈巧，而獵人則依賴敏捷和靈巧。職業切換時發生的素質重置消除了對單獨素質重置道具的需求，長期下來能節省 Zeny。

切換職業時，根據新職業的需求重新分配你的素質。每個職業的建議素質分配在我們的[職業構建區塊](/classes/all-classes-builds)中有詳細說明。錯誤的素質分配會顯著降低職業表現，因此在投入戰鬥前務必確認你的分配。

### 技能點規劃

每個職業都有自己獨立的技能樹，必須獨立發展。透過職業等級提升獲得的技能點僅適用於當前職業。當你切換離開一個職業，之後再返回時，你之前的技能分配會完全保留。這意味著你不需要為先前已發展過的職業重新分配技能點。

為每個職業仔細規劃你的技能點分配，以避免需要昂貴的技能重置道具。在投入點數前研究你的構建，並在投資通用技能前，先專注於定義你職業定位的核心技能。

## 職業自由與裝備

### 裝備倉庫策略

由於每個職業使用不同的裝備，當你維持多個職業時，你的倉庫會迅速被填滿。倉庫擴充系統提供了額外的儲存容量，而[序號獎勵](/codes/code-rewards-list)有時會包含倉庫擴充券。按職業整理你的倉庫，將每個職業的裝備分組存放，以便在切換時快速取用。

### 共享裝備

某些裝備類型可在多個職業之間共享，減少了你需要維護的物品總數。飾品、披肩和鞋子通常是全職業通用的，這意味著一件精煉良好的飾品能平等地惠及你所有的職業。優先精煉共享裝備物品，以在你的職業組合中獲得最大的投資回報。

## FAQ

### 我可以在戰鬥中切換職業嗎？

不可以，職業切換無法在戰鬥中或副本內進行。你必須在安全區域，通常是城鎮或沒有活躍戰鬥的野外，才能發起職業切換。在活動之間的自然空檔期規劃你的切換。

### 切換職業會重置我的任務嗎？

主線任務和已完成的支線任務進度會跨職業保留。但是，你正在切換離開的職業，其職業專屬任務進度會暫停，並在你切換回來時恢復。新職業開始時可以接取所有你基礎等級符合資格的任務。

### 實際上我可以練滿幾個職業？

大多數玩家會維持兩到三個達到或接近最高職業等級的職業。每個職業在裝備、精煉和技能發展上的資源投入，使得維持超過三個職業變得不切實際。在你的職業組合中，應注重品質而非數量。

### 職業切換有冷卻時間嗎？

職業切換本身沒有冷卻時間，但頻繁切換導致的 Zeny 成本增加，會作為一種實際的限制。每天多次切換職業的玩家會發現累積成本變得難以負擔。配合內容類型進行策略性切換，能將成本和不便降到最低。

### 我切換職業時，卡片會怎麼樣？

已插入裝備的卡片會跟隨該裝備。當你切換職業時，裝備及其卡片會移至倉庫。當你切換回原職業時，裝備和卡片會再次可用。共享裝備物品中的卡片對所有能裝備該物品的職業都有效。
