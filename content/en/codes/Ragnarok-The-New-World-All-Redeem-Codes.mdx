## How to Redeem Codes

Redeeming codes in Ragnarok The New World is a straightforward process, but there are a few important details to understand. The code redemption interface is accessible from the login screen as well as from within the game itself, giving you flexibility in when and where you enter your codes.

### Step-by-Step Redemption Process

1. Open the game and navigate to the Settings menu or the login screen gift code entry
2. Tap the "Redeem Code" or "Gift Code" button
3. Enter the code exactly as shown, respecting uppercase and lowercase letters
4. Complete the captcha verification that appears on screen
5. Confirm the submission and wait for the success notification
6. Check your in-game mailbox to claim the rewards

The captcha verification step exists to prevent automated code redemption bots from depleting the available code allocations. If you encounter difficulties with the captcha, try refreshing it or using a different device. The code entry screen on the login page allows redemption even before creating a character, which is useful for securing limited-time codes immediately upon starting the game.

### Troubleshooting Common Issues

If a code fails to redeem, check the following common causes: the code may have expired, it may be case-sensitive and incorrectly typed, your account region may not be eligible for that particular code, or the code may have a limited number of redemptions that has already been reached. Server-specific codes only work on the designated server, so verify the code is applicable to your server region. For persistent issues, contact customer support through the official Discord or in-game support system.

## FAQ

### How often are new codes released?

New codes are typically released every one to two weeks, coinciding with patch updates, community milestones, holiday events, and social media campaigns. Major content updates and anniversary celebrations often bring multiple codes simultaneously.

### Can I use codes on multiple servers?

Some codes are universal across all servers, while others are restricted to specific regions like SEA, TW, or the Steam server. The code description or announcement post will specify if a code is server-restricted.

### What happens if I enter an expired code?

Entering an expired code will display an error message indicating the code is no longer valid. You will not lose any items or resources, and the failed attempt has no penalty. Simply move on to trying other active codes.

### Are code rewards shared across characters?

Code rewards are typically bound to the account and can be claimed by one character per account through the mailbox system. Once claimed by a character, the items become character-bound and cannot be transferred to alternate characters.

### Where can I find the latest codes?

The best sources for new codes are the official Ragnarok The New World Discord server, the official Facebook page, YouTube channel announcements, and community fan sites like this one that maintain regularly updated code lists.
