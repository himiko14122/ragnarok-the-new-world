## Archer และ Hunter Build Guide

Archer และ Hunter เป็นตัวแทนของ DPS แบบโจมตีเดี่ยวต่อเนื่องสูงสุดใน Ragnarok The New World ด้วย Build ADL (Auto-Double-Launch) ทำให้ Hunter สามารถสร้างความเสียหายต่อเนื่องที่ไม่มีคลาสอื่นใดเทียบได้ในการต่อสู้ที่ยืดเยื้อ เหยี่ยวคู่ใจจะช่วยเสริมความเสียหายอัตโนมัติและมีประโยชน์ในการตรวจจับ สร้างคลาสที่โดดเด่นในการล่า MVP, การต่อสู้กับ Boss และการฟาร์มต่อเนื่อง ไกด์ Build นี้ครอบคลุมการจัดแต้ม ADL ที่ดีที่สุด, Build ทางเลือกแบบ Trapper และ Falconer, และกลยุทธ์การเลือกอุปกรณ์ที่เพิ่มศักยภาพ DPS ของ Hunter ให้สูงสุด

จุดแข็งของ Hunter อยู่ที่ความสม่ำเสมอ ในขณะที่คลาสที่สร้างความเสียหายแบบ Burst อย่าง Assassin จะปล่อยความเสียหายออกมาเป็นชุดใหญ่ในช่วงเวลาสั้นๆ Hunter จะรักษาสายธารแห่งความเสียหายอย่างต่อเนื่องผ่านการโจมตีปกติที่เสริมด้วย Double Strafe, การโจมตีอัตโนมัติของเหยี่ยว และโบนัสจากธนูธาตุ ความเสียหายต่อเนื่องนี้ทำให้ Hunter เป็นตัวทำดาเมจหลักในปาร์ตี้ล่า MVP ซึ่งการต่อสู้ยาวนานพอที่ความสม่ำเสมอจะเหนือกว่าการ Burst

## การจัดแต้มค่าสถานะ

### ADL Hunter Build

Build ADL เป็นรูปแบบ Hunter ที่ได้รับความนิยมและมีประสิทธิภาพสูงสุด โดยเน้น DPS ต่อเนื่องผ่านความเร็วโจมตีและความเสียหายคริติคอล

| Stat | Value | Purpose |
|------|-------|---------|
| STR | 1 | ไม่มีประโยชน์สำหรับดาเมจระยะไกล |
| AGI | 80-100 | ASPD เพื่อความถี่ในการโจมตีปกติสูงสุด |
| VIT | 20-40 | เพื่อความอยู่รอดขั้นต่ำ |
| INT | 1-10 | ตัวเลือกเสริม สำหรับดาเมจเหยี่ยวและ SP |
| DEX | 90+ | สถานะหลักสำหรับดาเมจโจมตีระยะไกล |
| LUK | 40-60 | อัตราคริติคอลสำหรับการโจมตีคริติคอลอัตโนมัติ |

DEX เป็นสถานะหลักในการเพิ่มดาเมจ เพราะมันส่งผลโดยตรงกับพลังโจมตีระยะไกลและความแม่นยำ AGI ให้ความเร็วโจมตีที่ทำให้ Build ADL มีประสิทธิภาพ เนื่องจากการโจมตีต่อวินาทีที่มากขึ้นแปลเป็นความเสียหายที่มากขึ้นโดยตรง LUK เสริม Build ด้วยอัตราคริติคอล เนื่องจากการโจมตีคริติคอลจะสร้างโบนัสความเสียหาย 40% ใน Ragnarok The New World การผสมผสานระหว่าง DEX สูง, AGI สูง และ LUK ปานกลาง จะสร้าง Hunter ที่โจมตีเร็ว, รุนแรง และติดคริติคอลบ่อยครั้ง

### Trapper Hunter Build

Build Trapper ยอมเสียสละความเสียหายจากการโจมตีปกติต่อเนื่องเพื่อความเสียหายแบบ AoE ผ่านสกิลวางกับดัก Build นี้เหมาะกับสถานการณ์เฉพาะในพื้นที่ฟาร์มที่รู้เส้นทางเดินของมอนสเตอร์แน่นอน

| Stat | Value | Purpose |
|------|-------|---------|
| STR | 1 | ไม่มีประโยชน์ |
| AGI | 40-60 | ASPD ปานกลางสำหรับวางกับดัก |
| VIT | 30-40 | ความอยู่รอดระหว่างการวางกับดัก |
| INT | 30-50 | SP สำหรับสกิลกับดักและเหยี่ยว |
| DEX | 70-80 | ความแม่นยำและดาเมจปานกลาง |
| LUK | 1 | ลงทุนน้อยที่สุด |

### Falconer Build

Build Falconer เน้น INT และ LUK เพื่อเพิ่มความเสียหายและอัตราการเกิดของเหยี่ยวให้สูงสุด การโจมตีอัตโนมัติของเหยี่ยวสร้างความเสียหายโดยไม่ขึ้นกับอาวุธของคุณ ทำให้เกิดรูปแบบการฟาร์มที่ไม่ต้องดูแลมาก

| Stat | Value | Purpose |
|------|-------|---------|
| STR | 1 | ไม่มีประโยชน์ |
| AGI | 60-70 | ASPD เพื่ออัตราการเกิด |
| VIT | 20-30 | ความอยู่รอดขั้นต่ำ |
| INT | 40-60 | ตัวคูณดาเมจเหยี่ยว |
| DEX | 70-80 | ความแม่นยำ |
| LUK | 30-50 | อัตราการเกิดของเหยี่ยว |

### Archer Skills (ก่อนเปลี่ยนอาชีพ)

| Skill | Points | Priority | Notes |
|-------|--------|----------|-------|
| Double Strafe | 10 | สำคัญมาก | สกิลดาเมจหลักที่ใช้บ่อย |
| Owl's Eye | 10 | สำคัญมาก | โบนัส DEX, มีผลตลอดเวลา |
| Vulture's Eye | 10 | สำคัญมาก | โบนัสระยะและแม่นยำสำหรับธนู |
| Improve Concentration | 10 | สูง | Buff AGI และ DEX, ใช้ตลอดเวลา |
| Arrow Crafting | 1 | ต่ำ | สร้างลูกธนูฉุกเฉิน |

Double Strafe เป็นสกิลหลักของ Archer ที่ใช้บ่อย โดยยิงโจมตีเร็วสองครั้งด้วยเวลาร่ายที่ลดลง ที่เลเวล 10 Double Strafe จะกลายเป็นหัวใจหลักของวงจร ADL เสริมการโจมตีปกติด้วยการโจมตีเพิ่มเติม Owl's Eye และ Vulture's Eye มอบโบนัส DEX และระยะโจมตีที่มีผลตลอดเวลา ทำให้เป็นสกิลที่ควรอัพก่อน

## กลไกของเหยี่ยว

### เหยี่ยวทำงานอย่างไร

เหยี่ยวคู่ใจของ Hunter จะโจมตีอัตโนมัติตามค่าสถานะ LUK และ INT ของคุณ การโจมตีปกติแต่ละครั้งมีโอกาสกระตุ้นให้เหยี่ยวโจมตี สร้างความเสียหายอิสระที่ปรับตาม INT เหยี่ยวยังมีประโยชน์ในการตรวจจับผ่านสกิล Detect ซึ่งเปิดเผยศัตรูที่ซ่อนตัว รวมถึง Assassin ที่พรางตัวใน PvP

ความถี่ในการโจมตีของเหยี่ยวขึ้นอยู่กับ LUK และความเร็วโจมตีของคุณ LUK ที่สูงขึ้นจะเพิ่มโอกาสที่เหยี่ยวจะโจมตีต่อการโจมตีหนึ่งครั้ง ในขณะที่ ASPD ที่สูงขึ้นหมายถึงการโจมตีต่อวินาทีที่มากขึ้น และเพิ่มโอกาสให้เหยี่ยวโจมตีได้มากขึ้น การทำงานร่วมกันนี้ทำให้การรวม AGI-LUK ใน Build ADL มีประสิทธิภาพเป็นพิเศษในการเพิ่มความเสียหายจากเหยี่ยวให้สูงสุด

### การเพิ่มประสิทธิภาพความเสียหายของเหยี่ยว

เพื่อเพิ่มความเสียหายของเหยี่ยวให้สูงสุด ควรลงทุนใน INT และ LUK ควบคู่ไปกับการลง DEX และ AGI หลักของคุณ สูตรความเสียหายของเหยี่ยวปรับตาม INT เป็นหลัก ในขณะที่อัตราการเกิดปรับตาม LUK Hunter ที่มี INT 50 และ LUK 50 จะสร้างความเสียหายจากเหยี่ยวได้มากกว่าผู้ที่มี INT 10 และ LUK 10 อย่างเห็นได้ชัด แม้ว่าคนหลังจะมี DEX ดิบที่สูงกว่าก็ตาม

## ลำดับความสำคัญของอุปกรณ์

### อาวุธและลูกธนู

อาวุธของ Hunter คือธนู ซึ่งได้รับประโยชน์จากการปรับค่าด้วย DEX และโบนัสจากลูกธนูธาตุ แตกต่างจากคลาสอื่นที่พึ่งพาความเสียหายจากอาวุธเพียงอย่างเดียว Hunter ยังได้รับประโยชน์จากการเลือกประเภทของลูกธนู ซึ่งเพิ่มความเสียหายธาตุให้กับการโจมตีแต่ละครั้ง พกพาลูกธนูหลายประเภทเพื่อให้ตรงกับจุดอ่อนธาตุของเป้าหมายต่างๆ

### อุปกรณ์ ASPD

ความเร็วโจมตีเป็นสิ่งสำคัญสำหรับ Build ADL เนื่องจากความเสียหายที่ทำได้สัมพันธ์โดยตรงกับจำนวนการโจมตีต่อวินาที ถุงมือ ASPD, อาหารเพิ่ม ASPD และอุปกรณ์ที่เพิ่ม AGI ล้วนมีส่วนช่วยให้ถึงขีดจำกัด ASPD ที่ 193 ซึ่งจะเพิ่มความเสียหายของคุณให้สูงสุด [ไกด์การตีบวกแบบปลอดภัย](/refining/safe-refining-guide) มีแนวทางการตีบวกธนูที่คุ้มค่า

### การเลือกการ์ด

การ์ดอาวุธสำหรับ Hunter ควรให้ความสำคัญกับโบนัสเปอร์เซ็นต์ความเสียหายระยะไกลและการ์ดเพิ่มดาเมจตามขนาดของศัตรู การ์ดประเภทเผ่าพันธุ์และธาตุจะเพิ่มความเสียหายต่อมอนสเตอร์ประเภทนั้นๆ ทำให้มีค่าสำหรับการฟาร์มเฉพาะจุดหรือการเผชิญหน้ากับ MVP ที่รู้ประเภทของเป้าหมายล่วงหน้า

## FAQ

### Build ADL ดีกว่า Trapper สำหรับการฟาร์มหรือไม่?

สำหรับการฟาร์มส่วนใหญ่ Build ADL มีความสม่ำเสมอและต้องการการเตรียมการน้อยกว่า Build Trapper Build Trapper สามารถมีประสิทธิภาพเทียบเท่าหรือดีกว่า ADL ในพื้นที่เฉพาะที่รู้เส้นทางเดินของมอนสเตอร์แน่นอน แต่ Build ADL ใช้ได้ดีทุกที่โดยไม่ต้องรู้แผนที่ สำหรับการเล่นทั่วไป แนะนำให้ใช้ Build ADL

### จะรักษา SP ในฐานะ ADL Hunter ได้อย่างไร?

Build ADL มีการใช้ SP ค่อนข้างต่ำเนื่องจากการโจมตีปกติไม่ใช้ SP มีเพียง Double Strafe และ Blitz Beat เท่านั้นที่ใช้ SP และใช้เป็นครั้งคราวมากกว่าต่อเนื่อง พกไอเท็มฟื้นฟู SP พื้นฐานและใช้เครื่องประดับฟื้นฟู SP สำหรับการเล่นระยะยาว ความต้องการ SP น้อยกว่า Wizard มาก ทำให้ Hunter พึ่งพาตัวเองได้มากกว่า

### ควรพกลูกธนูอะไรบ้าง?

ควรพกลูกธนูอย่างน้อยสามประเภท: Silver Arrows สำหรับประเภท Undead และ Demon, Fire Arrows สำหรับประเภท Plant และ Insect, และลูกธนูธรรมดาสำหรับใช้งานทั่วไป ลูกธนูธาตุเฉพาะอื่นๆ จะเพิ่มความเสียหายต่อ MVP บางตัว [ตารางจุดอ่อนของ MVP](/mvp-hunting/mvp-weakness-chart) จะช่วยคุณเลือกประเภทลูกธนูที่เหมาะสมที่สุดสำหรับการต่อสู้แต่ละครั้ง

### Hunter สามารถ Solo ดันเจี้ยนได้หรือไม่?

Hunter สามารถ Solo ดันเจี้ยนส่วนใหญ่ได้หากมีอุปกรณ์เพียงพอ แต่การเผชิญหน้ากับมอนสเตอร์จำนวนมากหรือกลไกของ Boss ที่รุนแรงจะท้าทายหากไม่มีปาร์ตี้ ความเปราะบางของ Hunter หมายความว่าความผิดพลาดในการจัดตำแหน่งจะนำไปสู่การตายอย่างรวดเร็ว สำหรับ [Time Corridor](/dungeons/time-corridor-guide) ขอแนะนำอย่างยิ่งให้มีองค์ประกอบของปาร์ตี้

### Hunter เทียบกับ Gunslinger สำหรับ DPS ในการล่า MVP เป็นอย่างไร?

Hunter ให้ DPS ต่อเนื่องที่สูงกว่าในการต่อสู้ที่ยาวนาน ในขณะที่ Gunslinger ให้ความเสียหายแบบ Burst ที่ปรับเปลี่ยนได้ผ่านการสลับอาวุธ สำหรับการต่อสู้กับ MVP ที่ยืดเยื้อ โดยทั่วไปแล้ว Hunter จะทำดาเมจได้มากกว่า Gunslinger สำหรับการต่อสู้ที่มีข้อกำหนดระยะโจมตีที่หลากหลาย ความยืดหยุ่นของ Gunslinger จะให้ความเสียหายที่สม่ำเสมอในทุกช่วง ดู [PvE tier list](/tier-list/pve-tier-list) สำหรับการเปรียบเทียบโดยละเอียด
