## Cara Ganti Class di Ragnarok The New World

Mengganti class di Ragnarok The New World adalah fitur inti dari sistem Job Freedom, yang memungkinkan karakter mana pun untuk beralih di antara delapan class yang tersedia tanpa membuat karakter baru. Panduan ini memberikan langkah-langkah tepat untuk beralih class, menjelaskan apa yang berubah dan apa yang terbawa, serta menawarkan saran strategis untuk merencanakan transisi class Anda guna meminimalkan sumber daya yang terbuang dan memaksimalkan potensi multi-class Anda.

Sistem Job Freedom mencerminkan filosofi desain yang menghargai fleksibilitas pemain daripada komitmen permanen. Tidak seperti MMORPG tradisional di mana pemilihan class mengunci Anda ke satu jalur perkembangan, Ragnarok The New World mendorong eksperimen dan keragaman peran melalui satu karakter. Namun, sistem ini menyertakan biaya yang berarti untuk mencegah peralihan sepele dan menghargai perencanaan yang matang.

## Proses Beralih Class Langkah demi Langkah

### Langkah 1: Persiapkan Peralihan

Sebelum beralih class, siapkan equipment dan inventaris Anda. Pindahkan semua equipment khusus class ke penyimpanan vault Anda, karena item ini tidak dapat digunakan oleh class baru. Catat alokasi stat Anda saat ini karena stat akan direset setelah beralih. Verifikasi Anda memiliki cukup Zeny untuk menutupi biaya peralihan, yang meningkat berdasarkan jumlah total peralihan class Anda.

### Langkah 2: Akses Antarmuka Peralihan

Buka menu karakter dan navigasikan ke bagian Job Freedom. Atau, kunjungi NPC peralihan class yang terletak di Prontera dekat pusat kota. Antarmuka menampilkan delapan class dengan progres Anda saat ini di masing-masing class. Class yang pernah Anda mainkan sebelumnya menunjukkan job level mereka saat ini, sementara class baru menampilkan status awal.

### Langkah 3: Pilih Class Baru Anda

Pilih class yang ingin Anda ganti dari opsi yang tersedia. Antarmuka menunjukkan pratinjau penampilan class, skill yang tersedia, dan batasan equipment. Tinjau detail ini sebelum mengonfirmasi pilihan Anda, karena peralihan memerlukan biaya Zeny yang meningkat seiring seringnya perubahan.

### Langkah 4: Konfirmasi dan Transisi

Konfirmasi pilihan class Anda dan bayar biaya peralihan. Model karakter Anda berubah menjadi penampilan class baru, stat Anda direset ke dasar, dan bilah skill Anda dikosongkan untuk mencerminkan status awal class baru. Transisi memakan waktu beberapa detik dan dapat dilakukan di area aman mana pun.

### Langkah 5: Alokasikan Ulang Stat

Setelah beralih, Anda harus mendistribusikan ulang poin stat Anda sesuai dengan persyaratan class baru. Setiap class memiliki distribusi stat optimal yang berdampak langsung pada performa. [Bagian build class](/classes/all-classes-builds) menyediakan alokasi stat yang direkomendasikan untuk setiap class dan tipe build. Alokasikan stat sebelum memasuki pertempuran, karena distribusi default tidak optimal untuk class mana pun.

## Apa yang Berubah Saat Anda Beralih Class

### Reset dan Perubahan

Saat Anda beralih ke class yang berbeda, elemen-elemen berikut berubah:

- Job level direset ke 1 untuk class baru
- Alokasi skill dihapus dan harus dipelajari kembali
- Poin stat direset untuk didistribusikan ulang
- Buff aktif dan efek makanan dibatalkan
- Quest khusus class saat ini dijeda

Reset job level adalah perubahan paling signifikan, karena mengharuskan Anda membangun kembali job level dari awal untuk class baru. Namun, setiap job level yang sebelumnya diperoleh pada class yang Anda tinggalkan akan dipertahankan. Jika sebelumnya Anda menaikkan Mage ke job level 30, beralih ke Swordsman, lalu beralih kembali ke Mage, job level Mage Anda kembali ke 30.

### Apa yang Tetap Sama

Elemen-elemen ini terbawa di antara peralihan class:

- Base level tetap tidak berubah
- Semua quest yang telah diselesaikan tetap selesai
- Equipment tetap berada di penyimpanan Anda
- Zeny dan semua mata uang dibagikan
- Isi penyimpanan vault dipertahankan
- Keanggotaan guild berlanjut
- Kotak surat dan hadiah tertunda tetap ada
- Mount dan pet terbawa

Pertahanan base level dan progres akun berarti karakter Anda terus maju bahkan saat menjelajahi class baru. Sistem mata uang bersama berarti Zeny yang diperoleh di satu class tersedia untuk dibelanjakan di class lain.

## Biaya Peralihan dan Ekonomi

### Struktur Biaya

Biaya peralihan mengikuti struktur yang meningkat yang dirancang untuk mencegah peralihan berlebihan sambil memungkinkan fleksibilitas yang wajar. Peralihan pertama memerlukan biaya nominal Zeny, membuatnya dapat diakses oleh pemain baru yang ingin mencoba class berbeda. Peralihan berikutnya meningkat biayanya, menciptakan batasan ekonomi lunak pada frekuensi peralihan.

| Nomor Peralihan | Perkiraan Biaya | Catatan |
|---------------|-----------------|-------|
| ke-1 | Gratis atau minimal | Peralihan perkenalan |
| ke-2 hingga ke-5 | Moderat | Terjangkau untuk pemain aktif |
| ke-6 hingga ke-10 | Signifikan | Memerlukan perencanaan |
| ke-11+ | Mahal | Mencegah peralihan sering |

Biaya yang meningkat mendorong pemain untuk menetap di dua hingga tiga class daripada terus-menerus beralih di antara delapan class. Desain ini selaras dengan investasi sumber daya yang diperlukan untuk setiap class, karena mempertahankan equipment penuh untuk kedelapan class akan sangat mahal.

### Menganggarkan untuk Peralihan

Jika Anda berencana mempertahankan beberapa class, anggarkan biaya peralihan Anda bersama dengan biaya perkembangan lainnya. Hitung total biaya dari frekuensi peralihan yang Anda rencanakan dan pastikan [Zeny farming](/zeny-farming/zeny-farming-guide) Anda mencakup biaya peralihan dan investasi equipment tanpa menguras dana refinement Anda.

## Peralihan Class Strategis

### Peralihan Berdasarkan Konten

Strategi peralihan paling efisien menyelaraskan pemilihan class dengan konten yang Anda mainkan. Gunakan class tempur Anda untuk konten [MVP hunting](/mvp-hunting/mvp-hunting-guide) dan dungeon, lalu beralih ke Merchant Anda untuk [manajemen perdagangan stall](/zeny-farming/stall-trading-guide) dan aktivitas ekonomi. Pendekatan ini memaksimalkan kekuatan setiap class sambil meminimalkan waktu yang dihabiskan untuk aktivitas di mana class tersebut kurang optimal.

### Peralihan Berdasarkan Progresi

Strategi lain melibatkan peralihan class berdasarkan kebutuhan progresi. Saat class utama Anda mencapai batas refinement yang membutuhkan investasi Zeny signifikan, beralihlah ke class farming seperti Mage atau Blacksmith untuk menghasilkan pendapatan secara efisien. Setelah Anda mengumpulkan sumber daya yang cukup, beralihlah kembali ke class utama Anda dan berinvestasi di tingkat refinement berikutnya.

### Peralihan Sosial

Komposisi party terkadang membutuhkan peran spesifik. Jika guild Anda membutuhkan Priest untuk perburuan MVP atau Knight untuk pertahanan War of Emperium, beralih untuk mengisi peran yang dibutuhkan memberikan nilai sosial dan akses party yang mungkin tidak ditawarkan oleh class DPS Anda. [Panduan Job Freedom](/guides/job-freedom-guide) mencakup strategi party multi-class secara detail.

## FAQ

### Berapa kali saya bisa beralih class?

Tidak ada batasan keras untuk peralihan class. Anda dapat beralih sebanyak yang Anda inginkan, tetapi biaya yang meningkat membuat peralihan sering menjadi semakin mahal. Sebagian besar pemain beralih class satu hingga tiga kali per hari, menyelaraskan peralihan dengan aktivitas konten yang berbeda.

### Bisakah saya beralih class di dalam dungeon?

Tidak, peralihan class memerlukan berada di area aman di luar pertempuran dan instance dungeon. Keluar dari dungeon atau kembali ke kota sebelum memulai peralihan class. Rencanakan peralihan Anda di antara aktivitas daripada di tengah konten.

### Apakah saya kehilangan progres class lama saya saat beralih?

Tidak, semua progres class dipertahankan. Saat Anda beralih dari suatu class, job level, alokasi skill, dan status kemajuan Anda disimpan. Saat Anda beralih kembali, semuanya kembali persis seperti yang Anda tinggalkan. Pelestarian ini memungkinkan peralihan dengan percaya diri tanpa takut kehilangan progres.

### Bagaimana jika saya beralih ke class yang belum pernah saya mainkan sebelumnya?

Beralih ke class baru memulai Anda di job level 1 tanpa skill yang dipelajari. Anda harus mendapatkan job experience melalui pertempuran dan quest untuk mengembangkan class baru. Proses leveling awal berjalan cepat karena base level Anda tetap tinggi, memberikan base stat yang kuat yang mengkompensasi skill job yang hilang selama job level awal.

### Bisakah saya menggunakan Job Freedom selama PvP?

Anda tidak dapat beralih class selama pertandingan PvP atau di dalam arena PvP. Pilih class Anda sebelum memasuki konten PvP. Di War of Emperium, class Anda terkunci setelah pengepungan dimulai. Lihat [panduan arena PvP](/pvp/pvp-arena-guide) kami untuk strategi pemilihan class dalam konten kompetitif.
