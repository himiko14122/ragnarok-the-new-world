## Panduan Lengkap Penukaran Kode

Sistem penukaran kode di Ragnarok The New World dirancang agar dapat diakses dari beberapa titik dalam klien game. Panduan komprehensif ini mencakup metode penukaran dari layar login dan dalam game secara detail, memberikan tips spesifik platform untuk pengguna PC dan mobile, serta membahas setiap error umum yang mungkin Anda temui selama proses penukaran. Baik Anda menukarkan kode pertama atau memecahkan masalah yang persisten, panduan ini siap membantu Anda.

Penukaran kode adalah fitur inti dari strategi keterlibatan komunitas Ragnarok The New World. Pengembang secara rutin mendistribusikan kode melalui saluran resmi, dan setiap kode memberikan manfaat nyata dalam game mulai dari Bound Zeny dan Adventure Coin hingga kosmetik eksklusif seperti penutup kepala DJ Poring dan pewarna fashion. Memahami cara menukarkan kode ini secara efisien memastikan Anda mendapatkan nilai maksimal dari setiap kesempatan, terutama selama periode peluncuran server ketika kode bernilai tinggi biasanya tersedia.

## Metode Penukaran di Layar Login

Layar login adalah tempat paling nyaman untuk menukarkan kode, terutama bagi pemain baru. Anda tidak perlu memiliki karakter atau bahkan menyelesaikan tutorial untuk memasukkan kode dari layar ini. Ini berarti Anda dapat mengamankan kode waktu terbatas segera setelah menginstal game, sebelum menghabiskan waktu untuk pembuatan karakter atau bermain.

### Langkah Detail di Layar Login

Saat Anda meluncurkan game dan mencapai layar login, cari area entri kode hadiah. Di perangkat mobile, ini biasanya berupa tombol kecil di dekat bagian bawah layar berlabel "Redeem Code" atau dengan ikon hadiah. Di versi PC melalui Steam, tombol muncul di lokasi serupa tetapi mungkin sedikit bergeser karena tata letak layar yang berbeda.

Setelah mengetuk tombol penukaran, kotak dialog terbuka dengan bidang input teks. Masukkan kode Anda di sini persis seperti yang diberikan. Kode di Ragnarok The New World bersifat case-sensitive, jadi "ROW666" dan "row666" diperlakukan berbeda. Pendekatan teraman adalah menyalin kode dari sumbernya dan menempelkannya ke bidang tersebut, yang menghilangkan risiko kesalahan ketik.

Setelah Anda mengirimkan kode, layar verifikasi captcha muncul. Selesaikan captcha dengan akurat untuk membuktikan Anda adalah pengguna manusia. Setelah melewati captcha, sistem memvalidasi kode Anda terhadap database server. Jika valid, pesan sukses muncul dan hadiah diantrekan untuk dikirim ke kotak surat dalam game Anda.

## Metode Penukaran di Pengaturan Dalam Game

Pemain yang sudah masuk ke dunia game dapat mengakses fitur penukaran kode melalui menu pengaturan. Metode ini secara fungsional identik dengan metode layar login tetapi lebih nyaman ketika Anda menemukan kode saat sedang aktif bermain.

### Menavigasi ke Antarmuka Penukaran

Untuk mencapai antarmuka penukaran dari dalam game, buka menu utama dengan mengetuk ikon menu atau menekan tombol Escape di PC. Arahkan ke opsi Pengaturan, lalu gulir melalui tab pengaturan hingga Anda menemukan bagian "Redeem Code" atau "Gift Code". Dialog entri kode dan proses captcha bekerja persis sama seperti versi layar login.

Salah satu keuntungan penukaran dalam game adalah Anda dapat segera memeriksa kotak surat untuk mengonfirmasi hadiah telah tiba. Kotak surat dapat diakses dari HUD utama, biasanya diwakili oleh ikon amplop. Ketuk kotak surat, pilih surat hadiah kode, dan klaim item untuk menambahkannya ke inventaris Anda.

## Tips Spesifik Platform

### Penukaran di PC (Steam)

Di versi PC, input keyboard membuat entri kode dan captcha lebih cepat daripada di mobile. Gunakan Ctrl+V untuk menempelkan kode dari clipboard Anda. Browser overlay Steam dapat berguna untuk mencari kode dengan cepat tanpa meninggalkan game. Jika Anda bermain dalam mode 120 FPS, UI mungkin terlihat sedikit berbeda, tetapi lokasi tombol penukaran tetap konsisten dengan tata letak antarmuka standar.

### Penukaran di Mobile (iOS dan Android)

Pengguna mobile diuntungkan oleh kemampuan untuk beralih antar aplikasi untuk menyalin kode dari media sosial atau situs komunitas. Di iOS, clipboard disinkronkan di seluruh perangkat jika Anda menggunakan ID Apple yang sama, memungkinkan Anda menyalin kode di komputer dan menempelkannya di ponsel. Pengguna Android dapat menggunakan fitur layar terpisah untuk melihat daftar kode di samping game, menyederhanakan proses salin-tempel.

## Panduan Resolusi Error

### Error "Invalid Code"

Error ini berarti kode yang Anda masukkan tidak cocok dengan kode valid apa pun di database server. Periksa kembali ejaan dan huruf besar/kecil setiap karakter. Verifikasi kode belum kedaluwarsa dengan memeriksa daftar kode terkini. Jika kode seharusnya valid, konfirmasi bahwa wilayah server Anda memenuhi syarat untuk kode tersebut. Beberapa kode yang dirilis untuk peluncuran regional tertentu, seperti kode peluncuran SEA, tidak berfungsi di server regional lainnya.

### Error "Code Already Used"

Setiap kode hanya dapat ditukarkan sekali per akun. Jika Anda melihat error ini, kode tersebut sebelumnya telah digunakan di akun Anda dan hadiahnya sudah dikirimkan. Periksa kotak surat dan inventaris Anda untuk mengonfirmasi Anda telah menerima itemnya. Jika Anda tidak dapat menemukan hadiahnya, hubungi dukungan pelanggan dengan detail akun Anda dan kode yang dimaksud.

### Error "Code Expired"

Kode kedaluwarsa tidak dapat ditukarkan lagi. Tidak ada cara untuk mengaktifkan kembali kode yang sudah kedaluwarsa. Untuk menghindari ketinggalan kode di masa mendatang, ikuti saluran komunitas resmi dan periksa daftar kode secara teratur. Kunjungi [halaman semua kode redeem](/codes/all-redeem-codes) kami untuk daftar kode yang masih berfungsi terkini.

### Kegagalan Captcha

Jika captcha berulang kali menolak jawaban Anda meskipun memasukkannya dengan benar, layanan captcha mungkin mengalami kesulitan teknis. Tunggu lima menit dan coba lagi. Menghapus cache game atau memulai ulang aplikasi juga dapat menyelesaikan gangguan sistem captcha. Pada kesempatan langka, sistem captcha offline selama periode pemeliharaan server, dalam hal ini Anda harus menunggu pemeliharaan selesai.

## FAQ

### Di mana letak tombol penukaran kode?

Di layar login, tombol penukaran muncul di dekat bagian bawah tampilan, biasanya berlabel "Redeem Code" atau menunjukkan ikon hadiah. Dalam game, itu ditemukan di dalam menu Pengaturan di bawah tab akun atau layanan.

### Apakah saya perlu membuat karakter terlebih dahulu untuk menukarkan kode?

Tidak, kode dapat ditukarkan dari layar login sebelum membuat karakter apa pun. Ini direkomendasikan untuk kode sensitif waktu yang mungkin kedaluwarsa sebelum Anda menyelesaikan pembuatan karakter dan tutorial.

### Apa itu Bound Zeny dari hadiah kode?

Bound Zeny adalah varian mata uang yang tidak dapat diperdagangkan dengan pemain lain atau digunakan di rumah lelang dan sistem kios. Fungsinya identik dengan Zeny biasa untuk pembelian NPC, biaya penyempurnaan, dan layanan Kafra. Hadiah kode biasanya menyediakan Bound Zeny daripada Zeny yang dapat diperdagangkan untuk mencegah eksploitasi ekonomi.

### Bisakah saya menukarkan kode selama pemeliharaan server?

Tidak, sistem penukaran kode offline selama periode pemeliharaan terjadwal. Tunggu pemeliharaan selesai sebelum mencoba menukarkan kode. Jadwal pemeliharaan diumumkan sebelumnya melalui Discord resmi dan pemberitahuan dalam game.

### Bagaimana cara menghubungi dukungan tentang masalah kode?

Kirim tiket dukungan melalui pusat bantuan dalam game atau server Discord resmi. Sertakan nama akun Anda, kode yang coba Anda tukarkan, pesan error yang diterima, dan tangkapan layar jika memungkinkan. Tim dukungan biasanya merespons dalam waktu 24 hingga 48 jam untuk pertanyaan terkait kode.
